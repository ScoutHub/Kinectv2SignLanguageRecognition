BodyHandFingerRecognition est la version stable du projet, affichant une modélisation propre et sans bug.


BodyHandFingerRecognition_VersionEnDev est une version du projet plus avancée : 

- Hand est composée des joints center / tip / thumb. 
- Reconnaissance des noms des doigts : méthode pour 5 doigts pour commencer. 
- Avancement sur la transformation des points 2D en 3D. 

Cette version n'est pas optimisée et n'est pas très stable. A compléter par la suite.
